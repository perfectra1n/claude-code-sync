Source Control Management abstraction layer.

Provides a unified interface for Git using CLI commands. Supports repository initialization, cloning, committing, pushing, pulling, and other common SCM operations through the scm::Scm trait. SCM (Source Control Management) abstraction layer.

Provides a unified interface for Git and Mercurial using CLI commands. Backend selection is controlled via the Backend enum.
