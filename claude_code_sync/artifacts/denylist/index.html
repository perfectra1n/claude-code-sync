claude_code_sync::artifacts

Module denylist 

Source
Expand description

Hardcoded never-sync rules for artifact copies.

These are enforced at the single copy chokepoint in both directions: on push nothing denied leaves the machine, and on pull a poisoned sync repo cannot plant a denied file (or escape ~/.claude). No user configuration is consulted here — there is deliberately no way to override this list.

Returns true when a ~/.claude-relative path must never be copied by artifact sync, checking every path component against the deny rules.
