Command handlers backing the claude-code-sync CLI subcommands.

Each handler owns one subcommand end-to-end: prompting, calling into the modules above, and reporting. The binary is a thin clap shell over these. Command handler modules

This module contains all command handler functions extracted from main.rs, organized by functionality area.
